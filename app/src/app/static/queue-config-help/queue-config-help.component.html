Postać configu:

Config musi być plikiem .py

Config musi zawierać komentarz #BOTS# wewnątrz listy lub innej kolekcji. Komentarz zostanie zastąpiony obiektami typu BotController z przekazanymi id poszczególnych botów.

Przykładowy konfig:

                    
    # config.py
    
    bots = [
        #BOTS#
    ]
    
    runs = 1000
