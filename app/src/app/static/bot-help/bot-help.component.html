Postać bota:

Nazwa pliku musi być zgodna z nazwą podaną przy dodawaniu bota

  • pojedynczy plik "nazwa bota".py
  • package pythonowy gdzie nazwa folderu jest również zgodna z nazwą bota jak powyżej (tak żeby się dało zaimportować import nazwabota)

Package lub plik musi zawierać klasę BotController , której konstruktor przyjmuje jeden argument: identyfikator bota. Klasa musi również implementować property name , które zwraca identyfikator bota podany do konstruktora.

Przykładowy bot:

# example_bot.py
    
    class BotController:
        def __init__(self, name):
            self._name = name
        
        @property
        def name(self):
            return self._name
