Ścieżki:

CONTROLLER_RELATIVE_PATH - ścieżka w projekcie gdzie umieszczane mają być boty

CONFIG_RELATIVE_PATH - ścieżka gdzie umieszczony ma być plik konfiguarcji

GAME_RELATIVE_PATH - korzeń, folder z którego można uruchomić grę

RESULTS_RELATIVE_PATH - ścieżka do folderu gdzie zapisywane są logi gry

RUN_COMMAND - komenda do uruchomienia gry

Przykład:

CONTROLLER_RELATIVE_PATH = Picipolo/bots
    CONFIG_RELATIVE_PATH = Picipolo/config.py
    GAME_RELATIVE_PATH = Picipolo/
    RESULTS_RELATIVE_PATH = Picipolo/results
    RUN_COMMAND = python game.py
